Getting started
---

	# Clone project
	git clone git@github.com:jontingvold/type-text-processor.git
	
	# Install yarn
	# https://yarnpkg.com/en/docs/install
	brew install yarn # Mac/brew
	
	# Install dependencies
	yarn install
	
	# Run
	yarn start
	
	# Package
	yarn dist
	
	# Upgrade dependencies (will probably break things)
	yarn upgrade
